OKR: Objectives & Key Results

- Objective: {do action} {about topic} {for amount} {during timeframe}. KPI: {details}. SMART: {details}.
- Key Result: {output, outcome, impact}. KPI: {details}. SMART: {details}

KPI: Key Performance Indicator

- Title: exact name to avoid ambiguity.
- OKR: relate to objective and key result.
- Scope: areas that are included/excluded.
- Owner: accountable person for the indicator.
- Formula: exact calculation of the indicator.
- Units: exacte measurement units.
- Source: exact data sources and inputs.
- Frequency: exact timing of measurment.
- Comments: notes, ideas, feedback, etc.

SMART critera:

- Specific: target one area clearly precisely.
- Measurable: quantify progress by using metrics.
- Actionable: able to start sustain complete.
- Relatable: relevant reasonable connected context.
- Timely: timing is favorable schedulable boundable.
